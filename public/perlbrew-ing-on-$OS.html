Perlbrew

Fork me on github!!

This page is list of short cheatsheet on how to prepare your system so after it runs this bash script:

bash \curl -L https://install.perlbrew.pl | bash eval "$(./perl5/perlbrew/bin/perlbrew init-in-bash)" perlbrew install perl-5.26.1 --notest

you get a usable perl-5.26.1 installation.

Debian and Ubuntu

bash sudo apt install -y build-essential curl perl

CentOS 7

bash sudo yum install -y gcc curl patch perl
