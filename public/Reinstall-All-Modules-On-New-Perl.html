How To Reinstall All Modules On New Perl

Sometimes when you install a new perl version, you want to put all the modules you were using on a previous version onto your new perl install.

One way to do this is to use the CPAN.pm autobundle feature to capture your old install:- bash perl -MCPAN -e autobundle

You willll end up with a bundle file written containing all installed modules, for instance: Wrote bundle file /home/davidp/.cpan/Bundle/Snapshot_2011_04_28_00.pm

You can then install that bundle on the new perl install, and job done:

perl -MCPAN -e 'install Bundle::Snapshot_2011_04_28_00'
